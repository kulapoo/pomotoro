# Audio Assets

This directory contains audio assets for the Pomotoro application.

## Directory Structure

- `notifications/` - Sound effects for phase transitions and alerts
- `background/` - Ambient background sounds for focus sessions

## Supported Formats

The audio system supports the following formats:
- MP3
- WAV
- OGG
- FLAC

## Default Assets

### Notification Sounds
- `bell.mp3` - Classic bell sound
- `chime.mp3` - Gentle chime
- `ding.mp3` - Simple ding sound
- `gentle-bell.mp3` - Soft bell tone
- `wooden-block.mp3` - Wooden percussion sound

### Background Ambient Sounds
- `rain.mp3` - Gentle rain sounds
- `forest.mp3` - Forest ambience with birds
- `ocean.mp3` - Ocean waves
- `white-noise.mp3` - White noise for focus
- `brown-noise.mp3` - Brown noise (deeper frequency)
- `cafe.mp3` - Coffee shop ambience
- `fireplace.mp3` - Crackling fireplace
- `thunderstorm.mp3` - Distant thunder and rain

## Adding Custom Assets

1. Place audio files in the appropriate subdirectory
2. Use the "Add Custom Audio" feature in the settings
3. Select the audio file and assign it to a category
4. The file will be added to the audio library

## Technical Notes

- Audio files should be optimized for file size
- Notification sounds should be 1-3 seconds long
- Background sounds should loop seamlessly
- Recommended volume levels are normalized to prevent audio clipping
